Класс LeanTweenType
Граф связей класса LeanTweenType:

Подробное описание

Pass this to the "ease" parameter, to get a different easing behavior

Example:
LeanTween.rotateX(gameObject, 270.0f, 1.5f).setEase(LeanTweenType.easeInBack);
